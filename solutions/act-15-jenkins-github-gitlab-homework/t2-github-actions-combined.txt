1. Combined GitHub Actions workflow YAML file located in:
.github/workflows/combined.yml

name: Combined CI
on:
  push:
    branches:
      - main

jobs:
  super-lint:
    name: Super Linter
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: github/super-linter@v5
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}

  docker:
    name: Build and Push Docker Image
    runs-on: ubuntu-latest
    needs: [super-lint]
    steps:
      - uses: actions/checkout@v3
      - uses: docker/login-action@v3
        with:
          username: ${{ secrets.DOCKERHUB_USERNAME }}
          password: ${{ secrets.DOCKERHUB_TOKEN }}
      - run: docker build -t ${{ secrets.DOCKERHUB_USERNAME }}/my-image:latest .
      - run: docker push ${{ secrets.DOCKERHUB_USERNAME }}/my-image:latest

2. Secrets configured in the repository:
DOCKERHUB_USERNAME
DOCKERHUB_TOKEN

3. Evidence of combined workflow run:
The workflow "Combined CI" ran successfully in GitHub Actions.
It contains two jobs:
- super-lint
- docker

Dependency:
The docker job runs only after super-lint completes (needs: [super-lint]).
